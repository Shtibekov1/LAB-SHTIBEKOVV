## Вопросы для защиты

### 1. Что значит «JS слабо типизированный язык»?
JavaScript позволяет неявно преобразовывать типы данных.  
Например: `'5' * 2 → 10`, `'5' + 2 → '52'`.  
Интерпретатор сам приводит значения к нужному типу при выполнении операций.

### 2. Что значит «JS язык с динамической типизацией»?
Тип значения определяется **во время выполнения**, а не при объявлении.  
Переменная может хранить значения разных типов в разные моменты.

### 3. Типы значений в JS. Оператор определения типа.
Примитивные типы:  
`number`, `bigint`, `string`, `boolean`, `undefined`, `symbol`, `null`.  
Сложный тип — `object`.  
Оператор определения типа: `typeof значение`.

### 4. Чем отличаются строки в "", '' и ``?
- `""` и `''` — обычные строки (разницы нет).  
- `` ` ` `` — шаблонные строки, поддерживают многострочность и подстановку через `${...}`.

### 5. Операнд и оператор.
**Оператор** — символ действия (`+`, `-`, `*`, `===`, `&&`).  
**Операнд** — значение, к которому применяется оператор.

### 6. Примеры унарных, бинарных и тернарного операторов.
- Унарный: `-a`, `typeof x`.  
- Бинарный: `a + b`, `x > y`.  
- Тернарный: `условие ? значение1 : значение2`.

### 7. Истинные и ложные значения в JS.
Ложные (`falsy`): `false`, `0`, `-0`, `0n`, `""`, `null`, `undefined`, `NaN`.  
Все остальные считаются истинными (`truthy`).

### 8. Циклическое исполнение кода в JS.
Циклы повторяют блок кода, пока выполняется условие.  
Основные виды: `for`, `while`, `do...while`, а также `for...of` и `for...in`.

### 9. Создание функций в JS. Сигнатура функции.
Функция может быть:
- **объявлена**: `function name(a, b) { ... }`
- **анонимна** (функциональное выражение)
- **стрелочная**: `(a, b) => a + b`

Сигнатура функции — это её имя и список параметров.

### 10. Параметры функции и аргументы.
- **Параметры** — переменные, указанные в объявлении функции.  
- **Аргументы** — фактические значения, передаваемые при вызове.

### 11. Концепт «раннего возврата» из функции.
Если условие не выполняется, функция может завершиться раньше:
```js
if (!user) return; // выход до конца функции
```
Это делает код чище и уменьшает вложенность.

### 12. Как можно перебирать элементы массива?
- `for (let i = 0; i < arr.length; i++)`
- `for...of`
- `forEach()`
- Методы `map`, `filter`, `reduce` также перебирают элементы.

### 13. Методы массива filter, map, reduce.
- `filter()` — отбор элементов по условию.  
- `map()` — преобразование каждого элемента.  
- `reduce()` — свёртка массива в одно значение (например, сумму).

### 14. Объекты. Создание, ключи, значения, свойства.
Объект — коллекция пар `ключ: значение`.  
Пример:
```js
const user = { name: "Yuka", age: 20 };
```
Создание через литерал `{}` или `new Object()`.

### 15. Как получить значение из свойства объекта?
- Точечная нотация: `user.name`  
- Квадратные скобки: `user["name"]`  
- Перебор: `for (let key in user) { ... }`  
- Методы: `Object.keys(obj)`, `Object.values(obj)`, `Object.entries(obj)`.
